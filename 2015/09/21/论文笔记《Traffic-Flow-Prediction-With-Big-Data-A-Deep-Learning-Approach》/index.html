论文笔记《Traffic Flow Prediction With Big Data:A Deep Learning Approach》

发表于 2015-09-21   |  

这篇比较新，今年刚发的paper。

Abstract

  • 考虑时间和空间的内在相关性
  • stacked autoencoder
  • 学习到generic traffic flow features
  • trained in a greedy layerwise fashion【现在的fashion应该是整体训练而不是layerwise吧？】

Introduction

ITS ： Intelligent Transportation System

  • traffic flow prediction大大的依赖于历史数据和实时数据
  • 目前所存在的prediction model都使用的是浅层模型，效果并不令人满意
  • 深度学习训练时不需要先验知识
  • 论文中的模型是用SAE学习generic traffic flow features，时间空间信息都被考虑在模型中

Literature Review

定义

$X_i^t$ ： i地点，t时刻的车流量

任务

给出i = 1,2,…,m; t = 1,2,…,T
预测出今后的某一时刻($t+ \Delta$)的车流量

历史

  • 最早1970s提出ARIMA model去预测短期内的高速公路车流量
  • 后来衍生出很多model，大致分为三类：parametric techniques/ nonparametric methods/ simulations

parametric techniques: time-series models, Kalman filtering models
nonparametric techniques: k-NN, ANNs
simulations: traffic simulation tools

  1. 其中广泛应用的模型是time-series methods，这里面大部分都是ARIMA的变种。

  2. 由于traffic flow的随机性和非线性特点，很多nonparametric methods被提出。
    【这里遗留了一个问题】：什么是参数法和非参数法？
    k-NN表现不错，但是并没有linear time-series method好
    Chang ： 基于k-NN非参数回归的动态多间隔模型
    El Faouzi ： a kernel smoother for the autoregression function
    Sun ： a local linear regression
    Bayesian network
    online learning weighted support vector regression（SVR）
    various ANNs
    还有很多杂交(hybrid)算法：combine several methods
    Tan ： MA+ES+ARIMA+NN
    还有很多很多……
    不过一句话，NNs的效果最好：good prediction power and robustness

Methodology

Autoencoder

  • activation function：sigmoid
  • cost function：mean square
  • 一大问题：如果hidden unit比input unit多的话，学出来的identity function，也就是说hidden layer的output完全是input，没有学到任何新东西。
  • 解决：实验证明，hidden unit大于或加上了稀疏限制时，不会出现这个问题。
  • 稀疏限制：KL divergence

SAE

结构图很简单，一句话总结：多个ae+logistic regression

Training Algorithm

Pretrain Phrase：greedy pretrain the deep network layer by layer
Fine Tuning：BP to tune the whole model’s params -> better result

Experiments

Data

  • 加利福利亚州的高速公路系统
  • 2013年的前三个月的weekdays数据：前两个月的数据作为training data，后一个月作为test data
  • 同一段路分两个方向，但是本文中三个月的数据都是单车道的

Index of Performance 性能指标

三个不同的性能指标：
mean absolute error 绝对误差的平均值：
mean relative error 相对误差的平均值：
root mean square error 平方误差的均值开根号：

Determination of the Structure of a SAE Model

  • 输入层的神经元个数
  • 隐藏层的个数
  • 每个隐藏层的神经元个数
  • 模型考虑了交通流的空间和时间关系
  • 时间：要预测t时刻的交通状况，就要用到前面时间段的交通状况$X^{t-1}$,$X^{t-2}$,…,$X^{t-r}$
  • 输入大小为mr
  • 输出的大小是m，m是高速路的个数

任务：预测15min，30min，45min，60min的交通状况

实验参数：

  • r：1-12 这个参数是控制预测时参考的过往数据多少，1代表只参考前一个时间段的交通状况
  • 隐藏层的神经元个数：[100:1000:100]
    然后用grid search搜索最佳参数。发现，不同的预测任务（15/30/45/60min），最好参数设置不一样。
    但是有一个结论是：最好的隐藏层个数在[2,5]之间，不能太少也不能太多。

Results

对比observed traffic flow（真实值）和predicted traffic flow（模型输出）结果。

  • 相似的traffic pattern
  • 对于heavy和medium traffic flow，模型预测的很好，但是在low就表现的不是太好。
    分析原因是：low traffic flow时，observed flow和predicted flow很小的一点差异都会引起一个大的relative error

对比BP NN/random walk/SVM/radial basic function NN:
这些方法都没有SAE好。
BP NN：88%-90%
SAE：几乎都>90%
